# 🎬 КиноВечер AI

Telegram-бот для подбора фильмов с использованием OpenAI.

## Возможности

- 🤖 Общение с ИИ
- 🎬 Подбор фильмов
- ⭐ Избранное
- 💾 SQLite
- ⚙️ Настройка через .env

## Установка

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Запуск

```bash
python bot.py
```

## Настройка

Создай файл `.env` на сервере:

```env
BOT_TOKEN=ваш_токен_бота
OPENAI_API_KEY=ваш_api_ключ
```
